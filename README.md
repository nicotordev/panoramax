# panoramax

Monorepo `pnpm` para construir `panoramax`, una app de discovery social de panoramas y eventos locales para Chile.

## Estructura

- `apps/`: workspaces de aplicaciones
- `docs/`: material de producto, research y estrategia

## Comandos

- `pnpm install`
- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm typecheck`
